Doc2Vec with classic ML classifier - reduced preprocessing¶

Gensim Doc2Vec model can be also used to prepare data for classification with either classic ML classifier or neural network. In such a configuration Doc2Vec performs encoding focused on the similarity of documents while classifier performs final classification. One of possible configuration for optimizing such a configuration is to optimize using the final classification performance as the criteria.

The most crucial design considerations related to such a dual model are:

  • the classifier selected to work with Doc2Vec encoded vectors - it should be fast, and with as least configuration required as possible (to reduce model the complexity)
  • the parameters of Doc2Vec model, among others important params are:
    • encoding vector dimension
    • encoding algorithm

In the following solution Logistic Regression was used as the final classifier

Modules and data preprocessing¶

  • some functions related to preprocessing were moved to external module to keep the notebook clean
  • some functions used were not included as class method, just to avoid the effort rtequired to make a class suitable for paralell processing

Doc2Vec + LogisticRegression - finding optimal params for Doc2Vec¶

The solution is based on Doc2Vec training with LogisticRegression classification after each epoch.

  • This approach was selected due to the fact that there is no heuristic which allows to predict the params of Doc2Vec, even a basic one like the vector size
  • In addition, Doc2Vec has no function that allows to break training in progress, so the model must be always performed for the fixed number of epochs.

So, to quickly find the reasonable set of hyperparameters three building blocks are necessary:

  • master loop for various paramsets for Doc2Vec
  • method which encapsulates classifier
  • callback class to be invoked after each epoch of Doc2Vec training

def perform_classification(d2v_model, train_data, test_data, eval_fraction=0.2):
    """
    Logistic regression encapsulating method. 
    Responsible for mapping data from Doc2Vec vectors and performing the classification.
    The imlementation below can be used to train and test of the classifier 
    with train data and test data or 
    with train data divided into train and evaluation fraction.
    :param d2v_model: trained Doc2Vec model
    :param train_data: oryginal train data (only target values are used)
    :param test_data: oryginal test data (only target values are used)
    :param eval_fraction: what part of train data is used for evaluation
    :return: accuracy score, 
             evaluation type : 
                'eval' if subset of train records was used for evaluation
                'test' - if test set was used for evaluation
    """
    eval_size=int(train_data.shape[0]*eval_fraction)
    train_size=train_data.shape[0]-eval_size
    test_size=train_data.shape[0]
    vector_size=d2v_model.vector_size

    if eval_size>0:
        eval_arrays=np.ones((eval_size, vector_size))
        eval_labels=np.ones(eval_size, dtype='int')
        for i in range(eval_size):
            eval_arrays[i] = d2v_model.docvecs[i]
            eval_labels[i] = train_data['Sentiment'].iloc[i]

    train_arrays = np.ones((train_size, vector_size), dtype='float')
    train_labels = np.ones(train_size, dtype='int')
    for i in range(0, train_size):
        train_arrays[i] = d2v_model.docvecs[eval_size+i]
        train_labels[i] = train_data['Sentiment'].iloc[eval_size+i]

    test_arrays = np.ones((test_size, vector_size), dtype='float')
    test_labels = np.ones(test_size, dtype='int')
    for i in range(test_size):
        test_arrays[i] = d2v_model.docvecs[i]
        test_labels[i] = test_data['Sentiment'].iloc[i]


    if eval_fraction>0:
        # use train_data for training and eval data fro evaluation
        clf = LogisticRegression(C=0.75,max_iter=500, solver='lbfgs',n_jobs=12)
        clf.fit(train_arrays, train_labels)
        y_pred = clf.predict(eval_arrays)
        score=accuracy_score(y_true=eval_labels, y_pred=y_pred)
        score_type='eval'
    else:
        # use train_data for training and test data fro evaluation
        clf = LogisticRegression(C=0.75,max_iter=500, solver='lbfgs',n_jobs=12)
        clf.fit(train_arrays, train_labels)
        y_pred = clf.predict(test_arrays)
        score=accuracy_score(y_true=test_labels, y_pred=y_pred)
        score_type='test'
    return score, score_type

class progress_callback(CallbackAny2Vec):
    def __init__(self, model_params, model_name:str, results_dictionary:dict, temp_results:dict, save_path=None, verbose=True):
        """
        Class used as a callback method for Doc2Vec. 
        For each iteration onlty data from best epoch is stored and written to a dictionary.
        Dataset from Doc2Vec model is stored as a pickle.
        :param model_params: params used during optimization iteration
        :param model_name: the name of the model (it can be any string, but must be unique)
        :param results_dictionary: the dictionary to store results in
        :param save_path: the dictionary to store results dictionary and data from best model
        :param verbose: wheather to print output message
        """
        self.epoch = 0
        self.no_improvement=0
        self.best_score=0
        self.best_epoch=0
        self.save_path=save_path
        self.verbose=verbose
        self.model_name=model_name
        self.model_params=model_params
        self.results_dictionary=results_dictionary
        self.temp_results=temp_results

    def on_epoch_end(self, model):
        score, score_type=perform_classification(d2v_model=model,train_data=preproc.train_data,test_data=preproc.test_data,eval_fraction=0.2)
        self.temp_results[(self.epoch)]=score
        if score>self.best_score:
            self.no_improvement=0
            self.best_score=score
            self.best_epoch=self.epoch
            vectors_filename=''
            if self.save_path:
                vectors_filename = self.save_path + self.model_name + \
                                   '_best_model.pkl'
                save_obj(model.docvecs, vectors_filename)
            self.results_dictionary[(self.model_name)]={
                **self.model_params,
                **{'Epoch':self.epoch,
                   'Best score':score,
                   'Score type':score_type,
                   'Vectors filename':vectors_filename}}
            if self.verbose:
                print('Epoch: {}, Score: {:.4f}, Best score: {:.4f}, Score type: {}, Best epoch: {}'.format(
                    self.epoch, score, self.best_score, score_type, self.best_epoch))
        else:
            self.no_improvement+=1
            if self.verbose:
                print('No improvement for {} times, Epoch {}, Score: {:.4f}, Best score: {}'.format(self.no_improvement, self.epoch, score, self.best_score))
        self.epoch += 1

def dict_product(d):
    keys = d.keys()
    for element in product(*d.values()):
        yield dict(zip(keys, element))
        
paramset = list(dict_product(d2v_params))
_, iternum=0, len(paramset)
for params in paramset:
    iter_start=dt.datetime.now()
    temp_results={}
    _+=1
    print(' '.join([':'.join([str(k), str(params[k])]) for k in params]))
    model_name='__'.join(['_'.join([str(k),str(params[k])]) for k in params])
    d2v_model=Doc2Vec(vector_size=params['vector_size'],
                      dm=params['dm'],
                      min_count=params['min_count'],
                      sample=params['sample'],
                      workers=MAX_WORKERS,
                      epochs=DOC2VEC_EPOCHS,
                      dm_mean=params['dm_mean'],
                      max_vocab_size=params['max_vocab_size'],

                        progress_callback(model_params=params, 
                                          model_name=model_name,
                                          temp_results=temp_results,
                                          save_path=OUTPUT_PATH, 
                                          verbose=False,
                                          results_dictionary=doc2vec_results)    

                      ])
    d2v_model.build_vocab(documents=preproc.train_corpus)

Searching for vectorsize¶

Initial wide search¶

  • In order to find the right vector size values from 50 to 500 were examined with interval 50
  • For each vector size 12 iterations were performed, 30 epochs each
  • in paralell two min_count values were examined: 2 and 3
  • the experiment was performed in fur independent sessions, but all of them used the same dataset, without shuufling between iterations
